Help

Getting Started

Welcome to the Machiavelli game! Here's how to get started:

  • Create a new game or join an existing one
  • Choose your country when joining
  • Learn the basic game mechanics

Game Mechanics

The game is played in phases:

  • Spring Movement
  • Spring Retreats
  • Spring Builds
  • Fall Movement
  • Fall Retreats
  • Fall Builds
  • Winter Adjustments

Unit Types

  • Army (A): Land military unit
  • Fleet (F): Naval military unit
  • Garrison (G): Defensive unit in cities

Orders

Each unit can receive orders:

  • Hold (H): Unit stays in place
  • Move (-): Unit moves to a new area
  • Support (=): Unit supports another unit
  • Convert: Unit changes type

Need More Help?

If you need more detailed information about the game rules or mechanics, please refer to the game documentation or contact the game administrator.
